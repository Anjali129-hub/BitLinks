# BitLinks

**BitLinks** is a URL shortening and link management app built with **Next.js** and **Tailwind CSS**.  
It allows users to create short links, manage them easily, and track clicks.

## Features

- Shorten long URLs into compact links
- Track link clicks and analytics
- Responsive and modern UI with Tailwind CSS
- Built with Next.js (React framework) for fast performance

## Tech Stack

- **Next.js** - React framework for server-side rendering
- **Tailwind CSS** - Utility-first CSS framework
- **Node.js** - Runtime environment
- **JavaScript/TypeScript** - Programming language

## Getting Started

### Prerequisites

- Node.js installed (v16+ recommended)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Anjali129-hub/BitLinks.git

2. Navigate to the project folder:
  
  cd BitLinks

3. Install dependencies:

  npm install

4. Start the development server:

  npm run dev
